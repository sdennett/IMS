IMS (Inventory Management System)
 
Purpose:

   Web based application for managing inventory which is also mobile device friendly.

Components:
   HTML
   Bootstrap CSS styling
   MySQL database 
   jQuery and Angular for dynamic presentation and data binding
   PHP
